# pypi_search New Features

## Console Options
  - `--no-pager | -p`
  - `--no-color | -c`

## Search Option
  - `--fuzzy | -z'
    - Fuzzy Search
  - '--sounds-like | -S'
    - Sounds Like Search
  - '--search-desc | -D'
    - Search Descriptions
    - Possibly limit to 500
  - `--search-full | -F`
    - Search Full Descriptions
    - Limit 100

## Configuration
  - ~/.config/pypi_search/pypi_search.conf
  - Create a term based UI for pypi_search confiuration
